Exports

There's no set formula for applying the csv, doc, pdf, or xls exports, but .btn-compact works best for display and the color class can be applied on a case-by-case basis.

The print button should always be text + icon and use .btn-small instead of .btn-compact.

base card

The plain .card class can have an avatar, but doesn't support the profile items, such as celebration and follow list.

flippable cards

Toggle a .flipped class to switch between the front and back of a card.

Toggle a .flipped-alt class to switch between the front and back of a card in the opposite direction. (See the profile card below for an example.)

Add a .card-flip class to see the card animate as it flips over.

large card

Add the .card-large class for a bigger avatar.

Feel free to use .btn-large on the action button if desired.

small card

Add the .card-small class for a smaller avatar.

Feel free to use .btn-small on the action button if desired.

profile card

Add the .card-profile class to allow the select checkbox, follow button, and celebration graphic to show up. The logic to determine whether or not these are included should live in your template.

This includes an example of the .flipped-alt behavior when clicking the celebration graphic. The card has two separate content blocks within the .card-back: .card-back-content and .card-back-content-alt.

NOTE IF YOU WANT TWO SEPARATE BACKS, ONE FOR EACH FLIP DIRECTION:
Due to the way CSS applies classes and transformations, you will need to do some manual show/hide in your JS to preserve the proper card back content when flipping to the front.

You can use .card-large and .card-small with your profile card.

filled up card-back

When there's too much to fit on the back, add .card-filled to the .card-back container for a little extra room.

This only becomes really apparent/useful with fixed-height cards described below.

selected card

Toggling the .selected class on your profile card will mark it so.

...and if you aren't using .card-profile the card is really selected.

fixed height cards

Add one of .card-short, .card-medium, or .card-tall to fix the height of the card (non-mobile only).
